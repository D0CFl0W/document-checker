# Руководство для фронтенда

## Назначение UI

Пользователь выбирает пять файлов (титул, задание, рецензия, нормоконтроль, антиплагиат). Фронт собирает **один ZIP** и отправляет его на бэкенд. После успешной загрузки бэкенд возвращает JSON; по кнопке «скачать отчёт» фронт запрашивает **PDF** с результатами разбора.

## Базовый URL API

В `static/script.js` задано:

```js
API_URL: "http://127.0.0.1:8000"
```

Для продакшена замените на реальный origin (тот же хост, с которого открыт сайт, либо отдельный API-домен). Учтите **CORS**: при другом origin FastAPI должен разрешить запросы с вашего фронта.

## Имена файлов внутри ZIP

В архив должны попасть **ровно пять** файлов с такими префиксами (расширение — по факту выбранного файла):

| Ключ в ZIP | Слот в UI |
|------------|-----------|
| `1_title.<pdf\|docx>` | Титульный лист |
| `2_task.<pdf\|docx>` | Задание |
| `3_review.<pdf\|docx>` | Рецензия |
| `4_norm.<pdf\|docx>` | Нормоконтроль |
| `5_antiplag.<pdf\|docx>` | Антиплагиат |

Расширение берётся из `file.name` пользователя (`zipExt` в `script.js`). Не подставляйте всем слотам `.docx`: иначе PDF попадёт в архив с неверным расширением (бэкенд умеет частично чинить это по сигнатуре файла, но корректные имена — норма).

## Эндпоинты

### `POST /upload-archive`

- **Тело:** `multipart/form-data`, поле **`archive`** — Blob/File, обычно `documents_bundle.zip`.
- **Успех (200):** JSON:

```json
{
  "message": "Файл успешно загружен",
  "saved_name": "<uuid>_<имя_архива>.zip",
  "original_name": "documents_bundle.zip"
}
```

Сохраните весь объект в памяти (например `uploadedFileInfo`) — он понадобится для скачивания отчёта.

- **Ошибка:** `4xx/5xx`, тело с `detail` (FastAPI).

### `POST /download-report`

- **Тело:** `application/json` — **тот же объект**, что вернул `/upload-archive` (поля `message`, `saved_name`, `original_name` обязательны для схемы `FileDownloadRequest`).
- **Успех:** бинарный **PDF**, `Content-Disposition` с именем вида `Report_<original_stem>.pdf`.
- **Ошибка:** `500` с текстом в `detail`.

## Главная страница

`GET /` отдаёт HTML из `templates/index.html` со статикой с префикса `/static/`.

## Проверка локально

1. Запустите FastAPI (см. [backend.md](backend.md)).
2. Откройте `http://127.0.0.1:8000/`.
3. В `script.js` выставьте тот же `API_URL`, что и у сервера.
